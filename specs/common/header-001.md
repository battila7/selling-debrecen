# Header 001: Start

## Context

Let's create a header for the site.

## Requirements

- The header should contain the following:
  - Logo that takes to the root when clicked.
  - The following menu items:
    - Ingatlanjaink
    - Kínálóknak
    - Keresőknek
    - Rólunk
  - A button to schedule a call with the text "Időpontfoglalás"
- The header should be styled as follows:
  - Transparent background by default.
  - The background turns black when scrolled down.
- In terms of responsiveness:
  - When on a small screen (mobile), the following should appear
    - Logo
    - Időpontfoglalás button
    - Button to open a drawer menu.
